# TabHere

**[English](https://github.com/scarletkc/TabHere/blob/main/README.md)** | **中文**

---

TabHere 是一个 Chrome 扩展，为网页中几乎所有可编辑区域提供 AI 自动续写补全功能。

## 开发

```bash
npm install
npm run build     # 输出到 dist/
npm run dev       # watch 模式
npm run package   # 打包发布
npm run version:set -- x.y.z  # 设置版本号
```

在 `chrome://extensions/` → 开发者模式 → “加载已解压的扩展程序”，选择项目根目录。
